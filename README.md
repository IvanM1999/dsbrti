# 🚀 DestinyServicesBR

Hub Operacional de Engenharia de Software, Metrologia Industrial e Ecossistemas Digitais.

**DestinyServicesBR** é uma plataforma integrada focada em desenvolvimento de software moderno, sistemas web interativos e integração com hardware/mecatrônica. Nasceu com o propósito de aproximar o desenvolvimento de software das necessidades reais da automação de hardware e do impacto comunitário.

---

## 📋 Sobre o Projeto

A DestinyServicesBR atua na intersecção entre a eficiência do código puramente digital e a robustez lógica exigida pelo chão de fábrica e sistemas de controle. Desenvolvemos:

- **Sistemas Públicos** com foco em acessibilidade e utilidade social
- **Plataformas de Triagem Automatizada** para saúde e proteção comunitária
- **Ferramentas de Metrologia Industrial** com conversores bidirecionais e cálculos de precisão
- **Landing Pages Comerciais** 
- **Ecossistemas Digitais** personalizados

---

## 🛠️ Tecnologias

- **Backend**: Node.js (HTTP Server com suporte a MIME types)
- **Frontend**: HTML5, CSS3, JavaScript moderno
- **Temática**: Sistema de temas dinâmicos (Ubuntu, Fedora, Windows XP)
- **Hospedagem**: Renderização local e cloud (Render)

## 🚀 Como Executar

### Pré-requisitos
- Node.js 14+ instalado

### Instalação

```bash
# Clone o repositório
git clone https://github.com/IvanM1999/dsbrti.git
cd dsbrti

# Inicie o servidor
node server.js
```

### Acesso

O servidor rodará em `http://localhost:3000` e redirecionará automaticamente para `http://localhost:3000/dsbrti/index.html`
