Grandezas Fundamentais
Grandeza Símbolo Fórmula Unidade
Carga Q - C (Coulomb)
Corrente I I = dQ/dt A (Ampère)
Tensão V V = W/Q V (Volt)
Resistência R R = V/I Ω (Ohm)
Potência P P = V·I = I²R = V²/R W (Watt)
Leis de Kirchhoff
KCL Σ I = 0 Correntes entrando = saindo
KVL Σ V = 0 Soma das tensões na malha
Resistores
Série Req = R1 + R2 + ... + Rn
Paralelo 1/Req = 1/R1 + ... + 1/Rn
Capacitores
Série 1/Ceq = 1/C1 + ... + 1/Cn
Paralelo Ceq = C1 + C2 + ...
Divisores
Tensão Vk = Vtotal × (Rk / Req)
Corrente Ik = Itotal × (Req / Rk)
Thevenin e Norton
Thevenin Vth, Rth
Norton In = Vth / Rth
Relação Rn = Rth
Dicas Rápidas
Resistores em série → soma
Resistores em paralelo → Req menor
Capacitores em série → Ceq menor
Capacitores em paralelo → soma
Malha → usar KVL
Nó → usar KCL
