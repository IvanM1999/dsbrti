Aula 7 — Revisão rápida para prova

Este é o resumo final. Se você souber responder estas perguntas, já está muito bem preparado.

Aula anterior Voltar ao início

1. Conceitos principais

  • O que é um CLP?
  • Qual a diferença entre entrada e saída?
  • O que é uma tag?
  • O que é uma FC?
  • Qual a função do Main?

2. Perguntas de revisão

  1. Em uma máquina, o botão é entrada ou saída?
  2. Uma válvula é entrada ou saída?
  3. O que significa uma porta AND?
  4. O que significa uma porta OR?
  5. Na linguagem Ladder, o que é a bobina?

3. Simulado rápido

  1. Explique com suas palavras o que faz um CLP.
  2. Descreva uma lógica simples de uma prensa.
  3. Explique por que usar tags faz a programação ficar mais organizada.

Dica final

Se você lembrar apenas desta frase, já vai bem: “Entrada percebe, lógica decide, saída age.”
