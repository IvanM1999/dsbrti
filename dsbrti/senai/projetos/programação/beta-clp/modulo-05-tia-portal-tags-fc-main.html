1. Tags

Tags são nomes para sinais. Elas tornam o programa organizado e mais fácil de entender.

2. FC

A FC é uma função onde você cria a lógica do processo. Ela separa o problema em partes menores.

3. Main

O Main é o bloco principal. Ele chama as lógicas que precisam ser executadas pelo CLP.

Desafio da missão

Liste três tags que você usaria em uma máquina simples e diga para que servem.

Resumo para prova

  • Tag = nome do sinal.
  • FC = bloco da lógica.
  • Main = bloco principal do CLP.
