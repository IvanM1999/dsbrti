Aula 5 — TIA Portal: criar tags, FC e Main

Chegou a hora de passar da teoria para a prática. No TIA Portal, você cria a estrutura do programa e organiza os blocos.

Aula anterior Próxima aula

1. O que são tags?

Tags são apelidos para os sinais do CLP. Em vez de lembrar de endereços, você usa nomes claros.

NomeTipoEndereçoFunção
Botao_IniciarBool%I0.0Botão de partida
Sensor_GradeBool%I0.1Proteção fechada
Valvula_PrensaBool%Q0.0Saída da prensa

2. Como criar uma FC

  1. Abra o Project Tree.
  2. Entre em Program blocks.
  3. Crie um novo bloco do tipo FC.
  4. Dê o nome Controle_Da_Prensa.

3. Como usar o Main

O Main é o bloco que o CLP executa automaticamente. Você deve chamar a FC dentro dele.

Passo final: arraste a FC para dentro do Main para que ela seja executada.

Resumo da aula

  • Tags organizam os sinais.
  • FC guarda a lógica.
  • Main executa a lógica.
