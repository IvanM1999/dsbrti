Aula 4 — Ladder: a linguagem do CLP

Agora a lógica deixa de ser abstrata e vira um desenho parecido com um circuito elétrico. Essa é a linguagem Ladder.

Aula anterior Próxima aula

1. O que é Ladder?

Ladder é uma linguagem visual. A corrente “anda” da esquerda para a direita. Se o caminho estiver livre, a bobina liga.

Regra: contatos abrem/fecham o caminho; bobina representa a saída.

2. Símbolos básicos

  • [ ] = contato aberto.
  • [/] = contato fechado.
  • ( ) = bobina.

3. Como montar a lógica

Botão iniciar -> grade fechada -> saída motor

Se o botão estiver pressionado e a grade estiver fechada, o motor liga.

Resumo da aula

  • Ladder é a linguagem visual do CLP.
  • Contatos controlam o caminho.
  • Bobina é a ação final.
