# Estrutura da versão Beta CLP

## Objetivo
Transformar o conteúdo de CLP em uma jornada didática mais envolvente para alunos iniciantes, com linguagem simples, profissional e preparada para prova.

## Diretrizes de conteúdo
- Falar como se estivesse explicando para alguém que nunca viu automação.
- Usar frases curtas, exemplos concretos e passos claros.
- Sempre separar: problema, lógica, programa e teste.
- Preparar cada aula para responder perguntas de prova.

## Gamificação inicial
- XP por missão concluída.
- Badge por módulo finalizado.
- Quiz curto ao fim de cada aula.
- Barra de progresso para mostrar evolução.

## Estrutura proposta
1. Módulo 0 — O que é CLP e por que aprender.
2. Módulo 1 — Entradas, saídas e pensamento lógico.
3. Módulo 2 — Fluxogramas e portas lógicas.
4. Módulo 3 — Ladder e relés.
5. Módulo 4 — TIA Portal, tags, FC e Main.
6. Módulo 5 — Motor, HMI/IHM e desafio final.
7. Módulo 6 — Revisão e simulado de prova.

## Próximo passo
Criar as páginas de cada módulo com esse mesmo padrão visual, mas com mais elementos de desafio, feedback e recompensa.
