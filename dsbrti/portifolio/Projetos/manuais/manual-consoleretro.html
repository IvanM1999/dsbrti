Manual Simples do Console Infantil

Transformando seu computador antigo em diversão segura

Passo 1 Passo 2 Passo 3 Passo 4 Diagnóstico

🔌 Passo 1: Preparar

Você vai precisar de:

  • Um pendrive (mínimo 8 GB)
  • Dois arquivos chamados “ISOs” (Lubuntu/MX Linux e Batocera)
  • Um programa para gravar esses arquivos no pendrive (Rufus ou Etcher)

Peça ajuda para alguém baixar os arquivos e colocar no pendrive. É como copiar fotos, mas com programas.

💻 Passo 2: Instalar o PC

Coloque o pendrive com Lubuntu/MX Linux e reinicie o computador.

Na tela, escolha Instalar. O sistema vai pedir para dividir o espaço do disco (pense como separar uma gaveta em duas partes):

  • Parte 1 → PC (Internet e YouTube)
  • Parte 2 → Console (Jogos)

Escolha “Instalação automática”. Depois, use o programa chamado GParted para ajustar.

🎮 Passo 3: Instalar o Console

Agora coloque o pendrive com Batocera e reinicie.

Na tela, escolha Instalar Batocera no HD e selecione a parte livre do disco.

Se não instalar, use o Batocera direto do pendrive. Funciona igual, mas sem ocupar espaço no HD.

🎨 Passo 4: Menu Inicial

Quando ligar o computador, vai aparecer um menu. Esse menu se chama GRUB.

Renomeie as opções para:

  • PC → Internet e YouTube
  • Console → Jogos retrô

Defina o Console como padrão, assim ele entra direto nos jogos.

Use o programa Boot Repair para reconstruir o menu. Se ainda não funcionar, entre na BIOS e escolha o sistema manualmente.
